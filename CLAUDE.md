## Contexto General
Este proyecto es una aplicación para almacenar el ciclo semanal de ventas y stock de una distribuidora de cigarrillos.

## Stack Tecnológico
- Framework: Next.js (App Router)
- Base de Datos: Supabase

## Reglas de Desarrollo
- Siempre usar TypeScript con interfaces explícitas.
